Outpost 2 Mission Software Development Kit

A conglomeration of C++ libraries easing the development of new Outpost 2 missions. Optimized for use with Microsoft Visual Studio.

The following submodules are included:

 * Outpost2DLL
 * OP2Helper
 * Hacker's Function Library (HFL)
 * odasl - Outpost 2 themed dialog skinning
