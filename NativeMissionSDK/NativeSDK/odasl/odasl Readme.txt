The odasl library allows easier skinning of dialog windows themed for Outpost 2.  Creating new dialogs for Outpost 2 may involve resources. Resource Hacker (http://www.angusj.com/resourcehacker/) is a great tool for reviewing or changing the resources of a DLL or executable.  

An article on creating a modal dialog box before a single player scenario can be found at: 
http://wiki.outpostuniverse.org/doku.php?id=op2_sdk:textdialogbox.
